# appmap-agent-js

JavaScript client agent for the AppMap framework

To install:
```sh
npm install @appland/appmap-agent-js
```

To run:
```sh
npx appmap-agent 
  [--cjs]
  [--esm]
  [--app-name my-app-name]
  [--map-name my-map-name]
  [--output-dir path/to/output/dir/]
  path/to/main.{js,mjs}
```
